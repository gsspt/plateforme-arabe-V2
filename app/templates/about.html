À propos de la plateforme

Explorez la langue arabe à travers sa morphologie et sa sémantique

🎯 Mission

La Plateforme Éducative Arabe V2 est un outil avancé d'analyse linguistique conçu pour les étudiants, chercheurs et passionnés de la langue arabe. Notre mission est de rendre accessible la complexité et la richesse morphologique de l'arabe à travers des technologies modernes.

🔧 Technologies

Backend

  • Python Flask
  • Base de données SQLite
  • Système de cache
  • Rate limiting

Analyse Linguistique

  • Transliteration Buckwalter
  • Analyse morphologique
  • Réseaux sémantiques
  • Base de données lexicale

Frontend

  • HTML5/CSS3 moderne
  • JavaScript ES6+
  • D3.js pour la visualisation
  • Design responsive

OCR

  • Google Vision API
  • Traitement d'images PDF
  • Optimisation pour l'arabe
  • Gestion RTL

📊 Fonctionnalités Principales

🔍 Analyseur Morphologique

Analyse complète des mots arabes : racines, formes verbales, dérivés et catégories grammaticales.

🌐 Réseau des Racines

Visualisation interactive des connections sémantiques entre les racines arabes.

📄 OCR Arabe

Reconnaissance de texte arabe dans les documents PDF avec Google Vision.

💾 Gestion des Données

Historique des recherches, favoris et statistiques d'usage.
